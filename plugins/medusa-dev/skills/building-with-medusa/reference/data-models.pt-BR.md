# Data Models

## Visão Geral

Modelos de dados definem a base do domínio. Eles devem refletir entidades reais e evitar sobrecarga de responsabilidades.

## Diretrizes

- use nomes claros;
- relacione entidades com parcimônia;
- prefira campos explícitos a blobs indefinidos quando o domínio exigir consulta estruturada.

## Casos de Uso Yello Solar Hub

- potência, fabricante, categoria técnica, distribuidor, status comercial, elegibilidade financeira.
