# Module Links

## Quando Usar Links

Use links quando duas entidades pertencem a módulos distintos e precisam se relacionar sem quebrar isolamento.

## Checklist

- definir link;
- migrar;
- consultar com query;
- validar comportamento de deleção e cascata quando necessário.

## Casos de Uso Yello Solar Hub

- fabricante ↔ produto;
- distribuidor ↔ oferta;
- cotação ↔ empresa.
