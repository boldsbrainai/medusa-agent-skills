# Custom Modules

## Quando Criar um Módulo Customizado

Crie um módulo quando o domínio exigir modelos, operações e ciclo de vida próprios.

## Estrutura

- `constants.ts`
- `models/`
- `service.ts`
- `index.ts`
- `migrations/`

## Checklist

- model definido;
- service registrado;
- módulo exportado;
- migration gerada e aplicada.

## Casos de Uso Yello Solar Hub

- `manufacturer`
- `distributor`
- `quote`
- `approval`
- módulos para bundles ou kits solares.
