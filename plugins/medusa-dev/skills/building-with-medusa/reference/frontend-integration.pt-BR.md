# Frontend Integration

## Visão Geral

O frontend deve consumir APIs Medusa com SDK e padrões de cache coerentes.

## Regras

- use SDK da Medusa sempre que possível;
- invalide queries após mutações;
- não replique regras de negócio no frontend.

## Casos de Uso Yello Solar Hub

- widgets admin para fabricantes;
- storefront B2B com kits solares e cotações.
