# Workflow Hooks

## Visão Geral

Hooks permitem estender workflows existentes sem fork da lógica original.

## Diretrizes

- conecte apenas ao fluxo certo;
- mantenha o efeito previsível;
- valide dados adicionais com clareza.

## Casos de Uso Yello Solar Hub

- associar fabricante ao criar produto;
- anexar dados comerciais ao criar proposta;
- disparar validação complementar em aprovação.
