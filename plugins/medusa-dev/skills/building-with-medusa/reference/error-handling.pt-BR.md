# Error Handling

## Visão Geral

Erros devem ser explícitos, consistentes e úteis para diagnóstico.

## Regras

- diferencie erro de validação, autenticação e negócio;
- retorne mensagens acionáveis;
- não esconda falha com sucesso parcial silencioso.

## Casos de Uso Yello Solar Hub

- fabricante inválido;
- distribuidor sem permissão;
- cotação com condição comercial inconsistente.
