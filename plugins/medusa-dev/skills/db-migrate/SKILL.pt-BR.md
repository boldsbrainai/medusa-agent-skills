# Executar Migrações de Banco de Dados

Execute o comando de migração do Medusa para aplicar migrações pendentes.

Use a ferramenta Bash para executar: `npx medusa db:migrate`

Reporte os resultados ao usuário, incluindo:

- Quantidade de migrações aplicadas
- Quaisquer erros que ocorreram
- Confirmação de sucesso
