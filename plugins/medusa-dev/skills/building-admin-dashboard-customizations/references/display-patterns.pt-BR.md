# Displaying Entities - Patterns and Components

## Visão Geral

Escolha o padrão de exibição conforme densidade, quantidade e tarefa principal.

## Opções

- `DataTable` para listas grandes;
- listas simples para coleções pequenas;
- cards quando o conteúdo exige mais contexto visual.

## Casos de Uso Yello Solar Hub

- DataTable para distribuidores e fabricantes;
- cards para kits solares ou bundles;
- listas rápidas para relações de produto.
