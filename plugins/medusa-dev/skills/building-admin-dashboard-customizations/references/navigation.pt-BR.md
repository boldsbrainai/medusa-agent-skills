# Navigation and Routing

## Visão Geral

A navegação do Admin deve conectar telas customizadas sem quebrar o fluxo operacional.

## Padrões

- use `Link` para navegação declarativa;
- use navegação programática quando houver fluxo pós-ação;
- leia parâmetros de rota de forma tipada.

## Casos de Uso Yello Solar Hub

- navegar da tela de produto para fabricante;
- abrir rota de cotação a partir de pedido;
- acessar detalhe de distribuidor a partir de uma listagem.
