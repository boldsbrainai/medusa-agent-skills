# Table Selection

## Visão Geral

Seleção em tabela é apropriada para datasets maiores e operações administrativas orientadas à escolha precisa.

## Diretrizes

- combine busca, paginação e seleção;
- deixe claro o estado atual da escolha;
- confirme ações destrutivas ou relevantes.

## Casos de Uso Yello Solar Hub

- escolher produtos relacionados;
- vincular fabricante a um conjunto de itens;
- selecionar distribuidores ou propostas em massa.
