# Typography

## Visão Geral

Tipografia no Admin deve priorizar legibilidade, hierarquia e consistência com o design system.

## Diretrizes

- use componentes tipográficos do sistema;
- reserve pesos fortes para títulos e labels críticos;
- use texto sutil para descrições auxiliares.

## Casos de Uso Yello Solar Hub

- destacar status de aprovação;
- separar metadados técnicos de textos de apoio;
- manter clareza em tabelas operacionais.
