# Forms and Modal Patterns

## Visão Geral

Formulários e modais no Admin devem ser previsíveis, rápidos e coerentes com a operação diária.

## Padrões

- `FocusModal` para criação;
- `Drawer` para edição contextual;
- desabilite ações durante mutação;
- mostre estado de loading e validação.

## Casos de Uso Yello Solar Hub

- criação de fabricante;
- edição de distribuidor;
- formulário de condição comercial ou aprovação.
