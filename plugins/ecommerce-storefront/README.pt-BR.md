# Plugin Claude Code para Ecommerce Storefront

Conjunto abrangente de orientações para construir storefronts de ecommerce com alta conversão, cobrindo UI, UX, SEO, responsividade e integração com backend.

## Instalação com Claude Code

1. Inicie o Claude.
2. Adicione o marketplace da Medusa.
3. Instale o plugin `ecommerce-storefront@medusa`.
4. Verifique se o plugin foi carregado.

## Uso

Ao pedir checkout, carrinho, listagem de produtos, busca, home, navegação ou integração com Medusa, o agente deve carregar esta base de práticas.

## Casos de Uso Yello Solar Hub

- Listagem de kits solares e equipamentos por fabricante.
- Checkout B2B com orçamento, revisão e pagamento.
- Busca por SKU, potência, fabricante e categoria técnica.
- Homepage com foco em distribuidores, financiamento e soluções corporativas.

## Privacidade

Este plugin não coleta nem transmite dados do usuário. O conteúdo instrucional é local.
