# Breadcrumbs

Orientações sobre implementação de breadcrumbs (trilha de navegação) para páginas de categoria e produto.

- Use estrutura semântica `<nav aria-label="breadcrumb">` com lista ordenada.
- Inclua o esquema de dados (JSON-LD) quando pertinente para SEO.
- Mostre o caminho hierárquico: Home → Categoria → Subcategoria → Produto.
- Em dispositivos móveis, permita truncamento com texto acessível completo em `title`.

Exemplo de marcação recomendada e práticas de acessibilidade.
