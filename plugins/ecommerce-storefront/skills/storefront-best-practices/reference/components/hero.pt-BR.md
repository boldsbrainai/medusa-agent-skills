# Hero

Padrões para seções hero (imagem, CTA e copy).

- Use imagens otimizadas e `loading="lazy"` abaixo da dobra.
- CTAs claros e visíveis; evite sobrecarregar com muitos botões.
- Forneça alternativas textuais e ARIA quando for um carrossel.
- Mantenha foco nos objetivos de conversão (ex.: promoção, assinatura).
