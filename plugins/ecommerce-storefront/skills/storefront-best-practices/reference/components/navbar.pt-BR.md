# Navbar

Padrões para barra de navegação: estrutura, acessibilidade e elementos essenciais.

- Inclua logo, busca, ícone do carrinho e menu de categorias.
- Certifique-se de navegação por teclado e foco visível.
- Em mobile, use um menu em drawer com acessibilidade adequada.
- Mantenha o cart count sempre visível e atualizado (`aria-live`).
