# Megamenu

Boas práticas para megamenu (navegação com categorias e destaques).

- Garanta posicionamento correto (`position: relative` no navbar).
- Evite fechamento automático ao navegar entre trigger e painel.
- Forneça versões mobile alternativas (drawer ou accordion).
- Use imagens reduzidas para destaque de produtos, sem afetar performance.
