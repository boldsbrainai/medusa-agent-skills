# Product Card

Padrões para cartões de produto usados em listagens e grids.

- Mostrar imagem principal, título, preço e call-to-action (Adicionar ao carrinho/Ver produto).
- Use `loading="lazy"` para imagens fora da dobra e otimize tamanhos.
- Indique badges (novo, promoção, estoque baixo) de forma consistente.
- Evite usar emojis; prefira ícones com texto alternativo.
