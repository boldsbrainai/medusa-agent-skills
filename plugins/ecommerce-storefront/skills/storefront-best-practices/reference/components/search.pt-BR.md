# Search

Diretrizes para barra de busca e autocomplete.

- Ofereça autocomplete com resultados principais (produtos, categorias).
- Use debounce (200-300ms) e destaque termos correspondentes.
- Permita busca por SKU, nome, atributos e categorias.
- Mostre estado de carregamento e mensagens de "Nenhum resultado" com sugestões.
