# Cart Popup (Mini Cart)

Boas práticas para o mini carrinho exibido em toda a loja.

- Atualize o contador com `aria-live="polite"` para leitores de tela.
- Mostre título, variação, quantidade e subtotal por item.
- Forneça ações claras: ver carrinho, finalizar compra, continuar comprando.
- Em mobile, use drawer ou bottom sheet com targets mínimos de toque (44px).

Indicadores de loading e comportamento ao adicionar itens.
