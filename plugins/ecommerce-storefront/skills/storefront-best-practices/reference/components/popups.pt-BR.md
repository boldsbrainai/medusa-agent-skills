# Popups

Diretrizes para popups e modais (promoções, newsletter, avisos).

- Evite popups intrusivos; respeite UX e frequência de exibição.
- Forneça controle de fechamento claramente acessível (Esc, botão Fechar).
- Não use popups para conteúdo crítico; prefira banners persistentes.
- Considere impacto em SEO e performance.
