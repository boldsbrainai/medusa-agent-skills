# Footer

Orientações para rodapé do site: organização, links e SEO.

- Inclua links essenciais: contato, políticas, FAQ, redes sociais.
- Use marcação semântica `<footer>` e listas para navegação.
- Otimize para mobile: colapsos por seção quando necessário.
- Não duplique conteúdo importante em múltiplos lugares para SEO.
