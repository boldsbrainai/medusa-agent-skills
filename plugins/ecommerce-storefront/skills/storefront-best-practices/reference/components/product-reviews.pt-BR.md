# Product Reviews

Padrões para exibição e submissão de avaliações de produtos.

- Mostre média de avaliação, contagem e comentários recentes.
- Para submissão, valide autenticação e trate spam/abuso.
- Use estrelas visuais com texto alternativo para leitores de tela.
- Considere moderação ou aprovação manual para conteúdos novos.
