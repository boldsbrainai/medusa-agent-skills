# Cart Page

## Visão Geral

A página de carrinho precisa dar confiança antes do checkout: itens corretos, preço legível, possibilidade de ajuste e próximo passo claro.

## Elementos

- Itens e variações.
- Atualização de quantidade.
- Resumo do pedido.
- Campo promocional quando aplicável.
- CTA para checkout.

## Diretrizes

- Recalcule totais após cada mutação.
- Mostre estados de loading e erro.
- Dê saída clara para continuar comprando.

## Casos de Uso Yello Solar Hub

- Compra conjunta de equipamentos principais e complementares.
- Conferência de volumes para integradores.
- Validação antes de seguir para fluxo B2B de aprovação ou pagamento.
