# Homepage Layout

## Visão Geral

A homepage deve orientar descoberta, reforçar proposta de valor e empurrar o usuário para os fluxos comerciais mais relevantes.

## Seções Essenciais

- Hero.
- Categorias em destaque.
- Produtos ou soluções em destaque.
- Provas de confiança.
- CTA para contato, orçamento ou catálogo.

## Hierarquia

- A proposta principal deve aparecer acima da dobra.
- Seções promocionais não devem competir com a navegação principal.
- O conteúdo deve apoiar busca por produto, solução ou serviço.

## Casos de Uso Yello Solar Hub

- Entrada para kits solares comerciais, distribuidores parceiros e financiamento.
- Bloco de fabricantes homologados.
- Seção para projetos empresariais e consultoria em tecnologia e energia.
