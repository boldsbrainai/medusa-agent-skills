# Order Confirmation Page

## Visão Geral

A confirmação de pedido reduz ansiedade pós-compra e orienta o próximo passo do cliente.

## Conteúdo Essencial

- Número do pedido.
- Resumo de itens.
- Endereço e entrega.
- Método de pagamento.
- Próximos passos.

## Diretrizes

- Reforce que o pedido foi recebido.
- Mostre CTA para acompanhar pedido ou voltar ao catálogo.
- Disponibilize contatos de suporte.

## Casos de Uso Yello Solar Hub

- Confirmação de compra de equipamentos para projeto empresarial.
- Orientação para contato comercial quando houver validação interna.
- Próximo passo para acompanhamento logístico ou documentação financeira.
