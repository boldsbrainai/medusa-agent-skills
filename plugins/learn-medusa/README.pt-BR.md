# Learn Medusa - Plugin de Tutorial Interativo

Tutorial guiado e interativo para aprender desenvolvimento com Medusa do zero, construindo uma funcionalidade real de brands enquanto a arquitetura é explicada passo a passo.

## Uso

Acione com prompts como:

- "Quero aprender Medusa"
- "Me guie no desenvolvimento com Medusa"
- "Ensine como construir features no Medusa"

## O Que Você Aprende

- Módulos customizados.
- Workflows com rollback.
- Rotas de API com validação.
- Module links.
- Hooks de workflow.
- Widgets e páginas do Admin.

## Casos de Uso Yello Solar Hub

- Adaptar o tutorial para fabricantes, distribuidores, kits solares e fluxos B2B.
- Ensinar o padrão `Module -> Workflow -> API Route -> Frontend` aplicado ao domínio solar.

## Privacidade

Todo o conteúdo instrucional é local e o MCP é usado apenas para documentação pública.
