# Lesson 3: Customize Admin Dashboard

## Objetivos de Aprendizado

Aprender como exibir e manipular dados de uma feature customizada dentro do Admin Medusa.

## Tópicos

- widgets;
- UI routes;
- SDK;
- React Query;
- padrões de carregamento e invalidação.

## Resultado Esperado

Ao fim, você deverá saber quando usar widget, quando usar tela dedicada e como conectar o Admin a APIs customizadas.

## Adaptação Yello Solar Hub

- widget de fabricante no produto;
- tela de distribuidores e ofertas;
- rota de cotações e acompanhamento comercial.
