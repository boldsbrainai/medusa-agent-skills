# Verification Guide

## Visão Geral

Depois de cada etapa, valide não apenas se "funcionou", mas se a solução respeita a arquitetura correta.

## Checklist de Verificação

- build sem erro;
- migration aplicada quando necessária;
- rota responde com payload esperado;
- mutação passa por workflow;
- frontend ou admin consome a API certa.

## Casos de Uso Yello Solar Hub

Use esse checklist para validar fluxos de fabricante, kits, distribuidores, aprovação e cotações B2B.
