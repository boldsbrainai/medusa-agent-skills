# Common Errors and Solutions

## Visão Geral

Erros são parte do processo de aprendizado. O objetivo deste guia é ajudar a identificar sintomas, causa raiz e correção sem pular entendimento.

## Categorias Comuns

- erros de módulo e registro;
- erros de workflow;
- erros de rota e autenticação;
- erros de Admin UI;
- erros de migração e banco.

## Estratégia de Diagnóstico

- peça o erro completo;
- verifique o comando executado;
- confirme o arquivo relacionado;
- revise se a etapa anterior foi concluída.

## Casos de Uso Yello Solar Hub

Os mesmos padrões de erro aparecem em módulos como `manufacturer`, `quote`, `approval` e `distributor`, então vale reutilizar o processo de diagnóstico para o domínio solar.
