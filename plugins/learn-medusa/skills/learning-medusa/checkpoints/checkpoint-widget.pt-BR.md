# Checkpoint: Widget

## Perguntas de Verificação

- O widget realmente estende uma página existente?
- A leitura de dados acontece na hora certa?
- O componente respeita o design system do Admin?

## Verificação de Implementação

- widget registrado na zona correta;
- dados carregam sem depender de modal;
- ações usam SDK e invalidam queries relevantes.

## Adaptação Yello Solar Hub

Use este checkpoint para um widget de fabricante no produto ou um resumo comercial em pedidos.
